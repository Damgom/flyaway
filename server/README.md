## 변동사항
- OAuth 인증 기능을 넣기에 시간이 모자랄 것 같아 일단 Session을 이용한 로그인서비스를 구현하고 차후 시간이 남는다면 구현해보는것으로 협의
- 테이블 명세서에서 따로 지정해뒀던 year, month, day column 부분을 YYYY_MM_DD 형식의 column으로 수정
- spring rest docs의 도입여부를 팀원과 상의 후 빼는것으로 결정
