🏋️‍♀️ FLYAWAY는 ‘운동’과 ‘기록’을 결합한 통합 웹서비스 입니다.

🎬 운동 관련 영상을 시청하면서 홈트레이닝을 진행해보세요!

📅 운동시간은 Flyaway에서 기록 해드리겠습니다!

🏆 쌓여가는 기록과 채워지는 캘린더를 보면서 성취감을 느껴보세요!

📄 자신의 운동 노하우나 식단, 일상들을 공유하며 함께 운동해보세요!

- FEAT : 새로운 기능의 추가
- REFACTOR: 코드 리팩토링
- TEST: 테스트 코트, 리팩토링 테스트 코드 추가
